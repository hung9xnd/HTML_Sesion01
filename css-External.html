What is CSS?

Cascading Style Sheets(CSS) is used to formart the layout of a webpage.

With CSS,you can control the color,fort,the size of text,the spacing between element,how elements are positioned and laid out,what background images or background colors are to be used ,different devices and screen sizes,and much more!

Tip:The workcascadingmeans that a style applied to a parent element will aloso apply to all children elements within the parent.So,if you set the color of the body text to "blue",all headings,paragraps,and other text element within the body will also get the same color(unless you specifi something else)!
